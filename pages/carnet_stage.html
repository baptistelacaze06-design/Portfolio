Carnet de Bord.

Application web de suivi de stage avec interface d'administration.

Voir le carnet en ligne ↗

Pour mon suivi de stage, j'ai conçu et développé un carnet de bord numérique sur-mesure. L'interface front-end a été intégrée en HTML/CSS pour un rendu clair et responsive. La véritable valeur ajoutée réside dans le back-end : j'ai développé un système d'authentification sécurisé en PHP permettant d'accéder à une interface d'administration. Depuis cette interface, il est possible de modifier dynamiquement le contenu du carnet, dont les données sont lues et sauvegardées de manière fluide dans un fichier JSON.
